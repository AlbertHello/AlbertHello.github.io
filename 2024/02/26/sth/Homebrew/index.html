Homebrew 安装与常用配置

本文最后更新于 2026-05-20T21:10:39+08:00

Homebrew是Mac上非常优秀的软件包管理工具。
由于国内的网络环境等问题，使用官方的安装方法可能无法安装成功。可以尝试使用其他更适合国内网络环境的安装方法，参考地址：https://brew.idayer.com/

安装homebrew

/bin/bash -c “$(curl -fsSL https://gitee.com/ineo6/homebrew-install/raw/master/install.sh)”

安装homebrew-core

cd "$(brew --repo)/Library/Taps/homebrew/"

git clone https://mirrors.ustc.edu.cn/homebrew-core.git

安装homebrew-cask

cd "$(brew --repo)/Library/Taps/homebrew/"

git clone https://mirrors.ustc.edu.cn/homebrew-cask.git

为了加速以后使用Homebrew安装其他软件的过程，建议设置软件源为国内源。

查看源

cd "$(brew --repo)" && git remote -v

cd "$(brew --repo homebrew/core)" && git remote -v

cd "$(brew --repo homebrew/cask)" && git remote -v

设置为中科大的源

git -C "$(brew --repo)" remote set-url origin https://mirrors.ustc.edu.cn/brew.git

git -C "$(brew --repo homebrew/core)" remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git

git -C "$(brew --repo homebrew/cask)" remote set-url origin https://mirrors.ustc.edu.cn/homebrew-cask.git

brew update 更新一下

设置bottles镜像

从macOS Catalina(10.15.x) 版开始，Mac使用zsh作为默认shell，使用的配置文件：.zprofile
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.zprofile

source ~/.zprofile

如果是以前的macOS版本，Mac使用bash作为默认shell，使用的配置文件：.bash_profile
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile

source ~/.bash_profile

重置为官方源

可以通过以下命令还原回官方源。

git -C "$(brew --repo)" remote set-url origin https://github.com/Homebrew/brew.git

git -C "$(brew --repo homebrew/core)" remote set-url origin https://github.com/Homebrew/homebrew-core.git

git -C "$(brew --repo homebrew/cask)" remote set-url origin https://github.com/Homebrew/homebrew-cask.git

zsh 注释掉 HOMEBREW_BOTTLE_DOMAIN 配置

vi ~/.zprofile

export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles

source ~/.zprofile

bash 注释掉 HOMEBREW_BOTTLE_DOMAIN 配置

vi ~/.bash_profile

export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles

source ~/.bash_profile

brew update

卸载

/usr/bin/ruby -e "$(curl -fsSL https://cdn.jsdelivr.net/gh/ineo6/homebrew-install/uninstall)"

注意

2022款Mac Book Air 13寸 M2芯片

  • brew 的位置在：/opt/homebrew/bin/brew
  • 通过 brew install 安装的库的可执行程序在/opt/homebrew/bin/ 不过这里的可执行程序是替身，原件在/opt/homebrew/Cellar/
  • 通过 brew install –cask 安装的软件在这里：
    /opt/homebrew/Caskroom

2019款Mac Book Pro 16寸 Inter芯片

  • brew 的位置在：/usr/local/bin/brew
  • 通过 brew install 安装的库的可执行程序在/usr/local/bin/ 不过这里的可执行程序是替身，原件在/usr/local/Cellar/
  • 通过 brew install –cask 安装的软件在这里：
    /usr/local/Caskroom
