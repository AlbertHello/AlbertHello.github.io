Mac 安装 Python 虚拟环境创建 Django 项目

发表于 2019-04-19 | 分类于 Mac环境相关

此安装过程针对Python3 MacOS

1、安装pip ，pip是Python的包管理工具,用于安装Python包。pip是“Pip Installs Packages”的递归缩写.

2、Python虚拟环境。它把软件所需的全部依赖和环境变量包装到一个文件系统中，与电脑中的其他软件隔离开。

5、cd python35Virtual 进入此文件夹。能看到自动生成了一些东西

7、激活后在安装需要的库。比如Djang。安装完成之后可以进入Python交互模式，导入Django试试有没有成安装.

8、使用cd .. 命令退到上一及菜单。然后选择一个你要创建项目的目录。比如进入桌面，创建项目。输入一下命令回车。没有反应才是好结果。
